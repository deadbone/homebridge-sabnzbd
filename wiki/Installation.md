# Installation

## Requirements

- Homebridge `^2.0.0`
- Node.js `^22 || ^24`
- SABnzbd stable 5.x API

## Homebridge UI

After npm publication:

1. Open Homebridge UI.
2. Go to **Plugins**.
3. Search for `homebridge-sabnzbd`.
4. Install the plugin.
5. Restart Homebridge.

## npm

```sh
npm install -g homebridge-sabnzbd
```

## Local Development

```sh
npm install
npm run build
npm link
```

Then add the `Sabnzbd` platform in your Homebridge configuration.
