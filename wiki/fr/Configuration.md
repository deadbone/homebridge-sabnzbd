# Configuration

Valeurs obligatoires :

- `url` : URL de l'interface web SABnzbd.
- `apiKey` : clé API SABnzbd disponible dans Config > General.

Les options règlent le timeout, la fréquence de rafraichissement, la pause temporaire, la limite de vitesse, la fenêtre des échecs récents et l'exposition de l'interrupteur d'effacement des avertissements.
